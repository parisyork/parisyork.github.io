Detailed Description

A simple mesh class.

each mesh consists of vertices, normals, texture coordinates each mesh can have ONE texture and/or COLOR

Remarks
current idea is to have the mesh take care of drawing itself. each contains in own VBO, VAO, these are generated by opengl calls so these classes cannot be copied
the mesh takes care of drawing itsself, the shader used must have vertex layout values match as below
Attribute Location
Position 0
Normal 1
Texture UV 3
for the OpenGL Draw call, default type is GL_TRIANGLES; this can be changed after initGL() has been called
default color is white with 1.0 alpha value, chosen because this will not interfere with texture sampler

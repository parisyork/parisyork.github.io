Detailed Description

Default Color is Black ( 1.0, 1.0, 1.0, 1.0) Default Style is Basic

Static Text usage: string, position, color are fixed

  1. Specify a map key, font file, font size and, optionally style
    add_font( "Lucida", "../Resources/fonts/LucidaConsole.ttf", 18, TextManager::Style::Basic );
  2. Create the message, location, assign color
    add_message( "Lucida", "Text String", X, Y, black );
  3. Render all text added to the text manager
    display_messages( time );

Changing Text usage, string, position, and color can change

Some or all of the letters change, such as FPS display

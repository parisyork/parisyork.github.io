FourD
A Method of Visualizing Four Dimensional Objects
In C++ using OpenGL

Introduction

Software for display of four dimensional information. The software will be a system to present four dimensional objects in an interactive and visually intuitive manner. Objects can be 4D analogs of actual 3D objects, such as cubes, ellipses or surfaces from multidimensional functions and data sets. The paradigm of computer aided design ( CAD ) will be used for its familiarity. CAD is the computer expression of three dimensional descriptive geometry, it is a direct approach to 3D object manipulation in two space. The four dimensional analog should provide some insight into the nature and shape of the four dimensional data sets.

The software will display three concurrent, connected projections of the 4D data set. The projections will be orthographic and parallel to the major axes â€“ X, Y, Z, and W. The user will interact with the system by manipulating a single projection at a time; the other projections will be updated in real-time reflecting the changes made.

This Project is implemented in C++ using OpenGL. Including libraries GLM FreeType ASSIMP ...

### Application description:
A CAD analog of three dimensional representation, extended into an additional spatial dimension. This is accomplished by having three viewing cubes, each representing three of four spatial dimensions. Cubes will necessarily be rendered as 2D for computer screen viewing. The object in the views will be linked. This allows rotation, zooming, and panning of the object in one view to be represented in the other two views. This interaction with the 3D representation of the 4D object will enable understanding of the shape and relation of faces, edges, and points of the object.
Initially only support for 4D polytopes is planned, hypercube, hypersphere. Data sets generated from functions, indexed points to display hyper-surfaces.

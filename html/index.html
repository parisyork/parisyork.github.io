FourD
A Method of Visualizing Four Dimensional Objects
In C++ using OpenGL

Introduction

Software for display of four dimensional information. The software will be a system to present four dimensional objects in an interactive and visually intuitive manner. Objects can be 4D analogs of actual 3D objects, such as cubes, ellipses, and other polytopes, surfaces, multidimensional functions, and indexed data sets. The paradigm of computer aided design, CAD, will be used for its familiarity. CAD is the computer expression of three dimensional descriptive geometry, it is a direct approach to 3D object manipulation in two space. The four dimensional analog should provide insight into the nature and shape of the four dimensional data sets.

Application description:

A CAD analog of three dimensional representation, extended into an additional spatial dimension. This is accomplished by having three viewing cubes, each representing three of four spatial dimensions. Cubes will necessarily be rendered as 2D for computer screen viewing. The object in the views will be linked. This allows rotation, zooming, and panning of the object in one view to be represented in the other two views. This interaction with the 3D representation of the 4D object will enable understanding of the shape and relation of faces, edges, and points of the object.

The software displays four concurrent connected projections of the 4D data, and one connected oblique projection. The projections are orthographic and from eye locations along the major axes â€“ X, Y, Z, and W. The user will interact with the system by manipulating a single projection at a time; the other projections will be updated in real-time reflecting the changes made.

Initially only support for 4D polytopes is planned, hypercube, hypersphere. Data sets generated from functions, indexed points to display hyper-surfaces.

User Interaction

Mouse Input:

Click and drag to rotate displayed data set, Arcball style. Quit application by selecting File->Quit

Keys Used:

  • Use Left/Right Arrow to move horizontally
  • Use Up/Down Arrow to move vertically
  • Use 0 to reset the position and scale for 3D
  • Use Left-Shift+0 to reset position and scale for 4D
  • Press W/E to rotate about the XW plane.
  • Press S/D to rotate about the XY plane.
  • Press X/C to rotate about the XZ plane.
  • Press R/T to rotate about the YW plane.
  • Press F/G to rotate about the ZW plane.
  • Press V/B to rotate about the YZ plane.
  • ...
  • ... more directions
  • ...
  • Press 4 to change 4D Eye Location to the XYZ
  • Press N to change 4D Eye Location to the NNN
  • ... encode action to each of 104 keys
  • Press Tab to cycle polytopes
  • Press ` to switch between single and multiple views
  • Press Esc to quit

Detailed Description

OpenGL 2D Texture class.

Create GL_TEXTURE_2D buffer from

  1. supplied data ( const unsigned char * )
  2. an image file
  3. single RGBA color, as vec4
  4. texture size, Rows x Columns, white in color
  5. OR just store a valid texture ID

It will then upload buffer to GL, manage create and delete of texture

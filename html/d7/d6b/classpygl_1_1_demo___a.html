Detailed Description

viewing of 4D data sets

Demo application for the viewing of 4D data sets.
opengl application that displays a single view or four connected views of a polytope, loaded as a CModel4 objects.
allows moving it with arcball and changing the 4D projection matrix among the four axes, X Y Z W. error with internal math library, uses GLM library.
using statements so that the switch between GLM and internal math library is not as complex

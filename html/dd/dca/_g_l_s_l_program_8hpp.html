Detailed Description

Author
Paris York
Date
4/29/15.

Basic OpenGL class to handle compiling and linking of shaders and related functions. OpenGL 4.0 Shading Language Cookbook, Wolff, David Also various online tutors. Designed for 3.2 Core functionality and related shader language, for MacOS and (hopefully) portability to POSIX linux, etc

Version
0.0 relies on the gl destructors, does not implement counting and releasing of shaders.
Does not implment copy and assignment operators. As written, shaders cannot be shared between programs.
to use the same shader in multiple programs, requires passing the source again to a different GLSLProgram after compiling, check for errors then clean up with glDeleteShader( GLuint shader ) once does with shader program clean up with glDeleteProgram( GLuint program )

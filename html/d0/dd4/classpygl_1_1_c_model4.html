Detailed Description

Four Space object to render

when drawing the four-space object

project from 4D camera to selected 3D space

GL Buffers do not support pushing <xyzw,1> create new buffer <xyw,1> or <yzw,1> or <xyz,1> or ...

Remarks
this is an OpenGL object, so shallow copy is impossible all gl object numbers need to be generated new and data reuploaded to new location

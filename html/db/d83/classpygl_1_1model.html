Detailed Description

Part of Spinning Cube

hold GL Model data, delete on exit. Necessary to render object: GL State, VAO Shader transforms model data buffer handle, VBO <– must be array of structs to only have one Buffer Object model texture handle, TexID

collection of meshes, texture, materials

a thing to be rendered by OpenGL, must have at least one mesh. each model class owns exclusively its own meshes, textures, and materials

§ draw()

void pygl::model::draw ( const programTime &  programTime)

Render the model to the screen, one frame

Parameters
programTimecurrent time
Precondition
must have shader selected and necessary uniforms set
uniform named gSampler for texture GL_TEXTURE0
uniforms named material.Ka, material.Kd, material.Ks, material.shininess

Render the model to the screen. the shader program is already bound and the per model instance/camera/projection matrices are set, then for each mesh

One Iteration

  1. bind the model Vertex Array Object
  2. activate GL_TEXTURE0 with texture OR uniform white texture
  3. if it has material, set associated uniforms
  4. set color uniform, vColor
  5. draw indexed buffer glDrawElements()
